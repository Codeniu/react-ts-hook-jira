## 项目配置

### json-server middware
1. 编写middleware.js文件
2. 注入到json-server中：
   更改package.json:
```shell
"json-server": "json-server __json_server_mock__/db.json --watch --port 3001 --middlewares ./__json_server_mock__/middleware.js"
```
